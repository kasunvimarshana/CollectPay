# Archived Reports

This directory contains historical reports that have been archived to maintain clarity in the main reports directory.

## Archive Policy

Reports are archived when:
1. They are superseded by newer comprehensive reviews
2. They represent intermediate progress (not final)
3. They are older than 30 days and not referenced in main documentation

## Current Active Reports

The following reports remain in the main `/documents/reports/` directory:

- Essential reference documents

## Accessing Archived Reports

All archived reports are preserved in git history and can be accessed:
- Through this archive directory
- Via git history: `git log --all --full-history -- <filename>`
- Through GitHub's file history view
